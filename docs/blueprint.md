# **App Name**: RSU iLEARN LMS

## Core Features:

- Teacher Command Center: Centralized dashboard for managing blocks, assignments, quizzes, and announcements.
- Student Link Portal: Secure portal for students to view assignments, track progress, and submit work via validated Google Drive URLs.
- Block Management: Tools for creating and managing class blocks.
- Automated Quiz Engine: Create, assign, and automatically grade quizzes with score calculation based on pre-defined criteria. Also implement AI scoring assistant as a tool.
- SpeedGrader with Split-Pane: High-density table with split-pane interface to preview student submissions (Google Drive ), interactive rubric, and comment bank.
- Rubric and Feedback System: Interactive Rubric tool that auto-calculates scores and a Comment Bank for providing rapid feedback.
- Role-Based Redirection: Firebase Auth with Google Sign-in, redirecting users based on their roles (Teacher/Student).

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to evoke a sense of trust and intelligence, appropriate for an educational setting.
- Background color: Light gray (#F0F2F5), offering a clean and neutral backdrop to reduce visual fatigue during prolonged use.
- Accent color: Teal (#009688) to provide a fresh and engaging highlight color for interactive elements.
- Body font: 'Inter', sans-serif. Headline font: 'Space Grotesk', sans-serif, for headlines. Inter is for body text.
- Code font: 'Source Code Pro' for displaying code snippets.
- Consistent, professional icon set to improve usability. Icons for different content types, file actions (download, etc.)
- Clean, grid-based layout that maximizes content density. Sidebar navigation for different blocks, filterable speedgrader.
